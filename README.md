# My Editor

## Project Description

Clone Coding Rich Text Editor

## v1.0.0

-   Bold, Italic, Underline, Strike out, Text Align
-   Heading (1, 2, 3, 4)
-   Ordered, Unordered List (ol, ul)
-   Block Quote
-   Code Block
-   Checkbox
-   Emoji
-   Image
-   Embed
