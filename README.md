# My Editor

## Project Description

Clone Coding Rich Text Editor.

## v1.0.0

-   Custom Formatting

    -   Bold, Italic, Underline, Strike out, Text Align
    -   Code Block

-   Custom Block

    -   Heading (1, 2, 3, 4)
    -   Numbered, Bulleted List (ol, ul)
    -   Block Quote
    -   Checkbox
    -   Emoji
    -   Image

    ## v.1.0.1 (TO-BE)

    -   Embed
