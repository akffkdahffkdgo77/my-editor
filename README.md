## Project Description

## v1.0.0 (2022.11.21 ~ 2022.12.04)

    -   Bold, Italic, Underline, Strike out, Text Align
    -   Code Block

-   Custom Block

    -   Heading (1, 2, 3, 4)
    -   Block Quote
    -   Numbered, Bulleted List (ol, ul)
    -   Checklist
    -   Image
    -   Emoji
